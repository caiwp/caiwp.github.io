Golang数组复制

2019-11-07 :: WenpiCai
#笔记  #Go 

参考链接 https://www.geeksforgeeks.org/how-to-copy-an-array-into-another-array-in-golang/

// creating a copy of an array by value
arr := arr1

// Creating a copy of an array by reference
arr := &arr1

两种方式结果不一样，第一种原值不变，第二种会导致原值也跟着改变
