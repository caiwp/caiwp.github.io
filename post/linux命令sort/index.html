Linux命令sort

2019-11-05 :: WenpiCai
#笔记  #Linux 

参考链接 https://www.cnblogs.com/coderchuanyu/p/4194211.html

忽略相同行使用-u选项或者uniq

　　
　　只显示单一行：
　　uniq -u file.txt 　
　　sort file.txt | uniq -u

　　统计各行在文件中出现的次数：
　　sort file.txt | uniq -c

　　在文件中找出重复的行：
　　sort file.txt | uniq -d
